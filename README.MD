# Panini World Cup 2026 Sticker Tracker 🌍

A web app to track your Panini World Cup 2026 sticker collection.

## Features
- 980 stickers organized by team and group
- Progress saved per user in the cloud
- Filters: all / collected / missing
- Live counter

## Tech stack
- Python + Streamlit
- Supabase (PostgreSQL)
- Deployed on Streamlit Cloud
